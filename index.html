Experimenting with NEAT for Stock Trend Prediction

My goal? To investigate NEAT's potential for predicting stock trends.

Technical Details

  • NEAT Library: Documentation
  • Dataset: Historical data for TSLA stock (Open, Close, High, Low).
  • Goal: Predict future trends (upward/downward) instead of absolute prices.
  • Network Input: Previous day's OHLC data yielded better results than past 5 days' closing prices.

Why NEAT?

While established models like SARIMA exist, NEAT offers a unique advantage. Its ability to evolve neural network architectures allows the network to automatically discover optimal configurations, potentially surpassing hand-crafted architectures.

Diving into NeuroEvolution of Augmented Topologies (NEAT)

A deep dive into the paper by Kenneth O. Stanley and Risto Miikkulainen.

Imagine this: You're training a robot arm to grab a cup. Using traditional methods, you tweak its "muscle" settings (weights) endlessly. With NEAT, you also evolve the arm's anatomy itself!

How NEAT Works

  • Historical Markings: Birthmarks for each part of the network to track origins.
  • Speciation: Grouping similar networks to encourage diversity.
  • Incremental Growth: Networks start simple and gradually add complexity.

The Big Deal

  • Robotics: Design robots that adapt to their environment.
  • Game AI: Train agents that strategize and learn tactics.
  • Creative AI: Generate art, music, and stories.

Advancements and Extensions

  • HyperNEAT: Optimizes the evolution process itself.
  • CPPN: Creates modular neural networks.

What is a VLM?

A Visual Language Model (VLM) is an advanced AI system designed to process and understand both images and text simultaneously, combining computer vision and NLP.

VLM in Moondream AI 0.5

The project integrates with Streamlit to create an interactive UI supporting:

  • General Image Description
  • Custom Question Answering
  • Point and Object Detection
  • Video Frame Analysis

Technical Implementation

Video Processing: Frame Extraction, VLM Analysis, Caption Generation, SRT generation.
